🏥 Hospital Management System (Complete App Information)
📌 Project Overview

A Hospital Management System (HMS) is a web application that helps hospitals manage patients, doctors, appointments, billing, medical records, and administration in one platform.

The system reduces paperwork, improves patient care, and streamlines hospital operations.

🎯 Main Objectives
Manage patient information
Book and track appointments
Manage doctor schedules
Store medical records
Generate bills and reports
Improve hospital workflow
Provide admin control over the entire system
👥 User Roles
1. Admin
Add/Edit/Delete doctors
Manage patients
View appointments
Manage departments
Generate reports
Manage billing
Control system settings
2. Doctor
View appointments
Access patient records
Write prescriptions
Update diagnosis
Manage availability
3. Patient
Register/Login
Book appointments
View medical history
Download prescriptions
Pay bills online
