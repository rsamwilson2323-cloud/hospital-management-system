# 🏥 Hospital Management System

A modern **Hospital Management System** built with **Node.js, React, TypeScript, and Tailwind CSS**. Easily manage patients, doctors, appointments, billing, pharmacy, medical records, and laboratory reports from one professional dashboard. Designed to work completely offline on your local network with **no database required**—all data is securely stored as local JSON files.

---

* Add, edit and delete patients

* Blood group tracking
* Emergency contact management
* Medical history records

### 🩺 Doctor Management

* Doctor profile management
* Specialization tracking
* Qualification & experience details
* Availability scheduling
* Doctor search and filtering

### 📅 Appointment Management

* Schedule appointments
* Reschedule appointments
* Cancel appointments
* Appointment status tracking

* Doctor & patient linking

* Diagnosis management

* Bill generation

* Recent activities

* Revenue reports

* Performance statistics
* Interactive charts

* Local JSON data storage
* No database required

* Local network access
* Mobile friendly

## ⚙️ Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/rsamwilson2323-cloud/hospital-management-system.git
```

### 2️⃣ Navigate to the Project Folder

```bash
cd hospital-management-system
```

### 3️⃣ Install Dependencies

```bash
npm install
```

---

## ▶️ Running the Application

### Option 1 – Windows Launcher

Double-click:

```text
start.bat
```

### Option 2 – Command Line

```bash
npm run dev
```

or

```bash
npm start
```

## 🌐 Access the Application

### Localhost

```text
http://localhost:5000
```
